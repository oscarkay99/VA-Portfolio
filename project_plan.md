# Virtual Assistant Portfolio

## 1. Project Description
A personal portfolio website for a Virtual Assistant / Operations professional. The site highlights key strengths (organization, attention to detail, proactive communication, systems thinking) and positions the owner as a reliable, professional VA ready for hire. Target audience: business owners, entrepreneurs, startups looking for remote operational support.

## 2. Page Structure
- `/` - Home (single-page portfolio with anchored sections)
  - Hero
  - About
  - Skills & Services
  - Why Work With Me
  - Testimonials
  - Contact / CTA
  - Footer
